Major Performance Challenge

Overcame Major Performance Challenge – 5000+ PDFs Handling

The Problem

The client had 5000+ PDF files that needed to be displayed in a table format inside Shopify.

Uploading 5000 PDFs directly to Shopify/WordPress causes:

  • Very slow loading
  • Heavy storage usage
  • Backend timeout issues
  • Poor SEO and poor user experience
  • Shopify also has file upload limitations

Solution Implemented (Using Google Drive + GCP)

I designed a scalable, high-performance solution:

Step 1 – Store 5000+ PDFs in Google Drive

Cloud storage instead of hosting PDFs in Shopify. Reduces theme size and avoids server slowdown.

Step 2 – Serve PDF Links via Google Cloud Platform (GCP)

Used GCP to generate secure, reliable, fast-loading public URLs. Eliminated the need for storing heavy files on Shopify.

Step 3 – Display PDFs in Shopify Table (10 Per Page)

Implemented server-side pagination (10 records per page). Prevents loading all PDFs at once. User can navigate page-by-page without lag.

Step 4 – Implement Lazy Loading

PDFs load only when the user scrolls down or switches pages. Improves loading speed by more than 80%. Helps keep performance scores high.
